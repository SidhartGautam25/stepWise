# Step 1: Return 42

## What to build

Export a function that returns the number `42`.

## Starter code

Open `index.js` in your workspace folder. You'll find a placeholder function:

```js
module.exports = function returnFortyTwo() {
  return null;
};
```

Change `return null` to `return 42`.

## How to test

Run `npx stepwise test` from inside your workspace folder.

## What the test checks

- The exported function returns exactly `42`

## Hint

`42` is a number, not a string. `return "42"` will not pass.
