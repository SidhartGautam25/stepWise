# Step 2: Sum Two Numbers

## What to build

Export a function that takes two numbers `a` and `b` and returns their sum.

## Starter code

Open `index.js` in your workspace folder:

```js
module.exports = function sumTwo(a, b) {
  return 0;
};
```

Replace `return 0` with the correct implementation.

## How to test

Run `npx stepwise test` from inside your workspace folder.

## What the test checks

- `sumTwo(2, 3)` returns `5`
- `sumTwo(-1, 4)` returns `3`

## Hint

The `+` operator in JavaScript adds numbers when both arguments are numbers.
