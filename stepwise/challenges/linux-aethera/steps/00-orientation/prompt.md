# Welcome to Intro to Linux

This lesson uses a small visual filesystem so you can see what your terminal commands do.

How to read the screen:

- The label at the top right of the visualizer is your current directory.
- The left side shows root directories such as `/bin`, `/etc`, `/tmp`, and `/home`.
- The main area shows the files and directories inside your current directory.
- Each tile shows type, owner, permissions, and file content when available.
- Green terminal output means a command worked.
- Red terminal output means the command failed and Linux rejected it.

Type `continue` in the terminal to begin.
